# asio-framework

### Requirements:
* conan
---  
### Build:
CMake options:
* -DCMAKE_BUILD_TYPE=[Debug/Release] | Choose what you need
* -DENABLE_TESTING=[ON/OFF]          | Enable tests or not

```bash
# Example of build in Debug mode with tests
cmake -B build -DCMAKE_BUILD_TYPE=Debug -DENABLE_TESTING=ON
```
